根据<[🍭 ADB 用法大全](https://github.com/mzlogin/awesome-adb)>这篇文章,把一些常用的命令做了封装.

### 运行
需要python3
```
python3 you-path/adb_command.py
```

### 功能
- 查看设备信息
- 查看包列表
- 查看apk信息(支持模糊查询)
- 查看ip
- 截图(截屏,上传文件,删除)
- 截屏(录屏,上传文件,删除)
